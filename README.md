Just my personal neovim config with Lazy plugin manager, telescope, LSP and few other changes accocding to my preferences.

# Setup
git clone https://github.com/Sarang-13/nvim_config.git ~/.config/nvim

# Dependencies
* clangd
* ripgrep
* fd
* node
* python3

```
sudo apt update

sudo apt install -y \
    neovim \
    ripgrep \
    fd-find \
    clangd \
    python3 \
    nodejs \
    npm
```
Append with #!/bin/bash to make a script for convinience
